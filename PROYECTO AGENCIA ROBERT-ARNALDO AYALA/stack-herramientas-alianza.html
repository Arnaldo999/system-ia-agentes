¿Qué IA usan los agentes de clientes?

Los agentes que entregamos a clientes usan Gemini (Robert), NO Claude.

  • Claude API = herramienta de desarrollo de Arnaldo (como un IDE potenciado). No genera costos en proyectos de clientes.
  • Gemini API = IA de producción en los agentes. Más económico, suficientemente potente para la mayoría de casos.
  • OpenAI = disponible cuando el cliente lo requiera explícitamente o el caso lo justifique.
  • ElevenLabs = síntesis de voz para agentes telefónicos. Probablemente el servicio más costoso del stack — solo se activa en proyectos que incluyan agentes de voz (Vapi).

Flujo de un Proyecto

1

Robert consigue el cliente, hace el cierre comercial y envía el brief a Arnaldo.

2

Arnaldo construye el sistema (bot, workflows, backend) usando Claude Code como herramienta de desarrollo.

3

Arnaldo deploya en la infraestructura, gestiona las APIs de IA y entrega el sistema funcionando.

4

Robert factura al cliente y gestiona la relación comercial. Arnaldo da soporte técnico continuo.
