## 🧠 Aprendizajes / Patrones nuevos

1. **Airtable reverse-link** — `filterByFormula` en linkedRecord = trampa. Usar el array reverse-link del padre. Ver `wiki/conceptos/airtable-reverse-link-pattern.md`.
2. **Upload pipeline FastAPI→Airtable** — Airtable requiere URL pública. Patrón: `/uploads/` static + URL. Ver `wiki/conceptos/fastapi-upload-attachment-pipeline.md`.
3. **Dark mode CSS vars** — `[data-theme="dark"]` + override `[style*="background: white"]` para estilos inline hardcoded.
4. **Airtable typecast en singleSelect** — usar `typecast: True` al crear para valores nuevos. PATCH directo sobre campo no existente = 422.
